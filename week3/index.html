Chapter 5 - JavaScript: Novice to Ninja

  • If the property value of an object is a function, it is called a method.
  • Object can be used to encapsulate code so it can be reused and it can inherit code from other objects,
  • Object literals are useful in keeping the code clean and not clogging down the namespace.
  • All objects are mutable at any time when a program is running even if it was declared using const.
  • You can declare an object using '{}' or you can use the object constructor. '{}' are the preferred method.
  • You can access a property using either dot notation or with brackets. Either method is fine.
  • A method can be called the same way a fucntion is called.
  • The in keyword can be used to see if an object has a property. It can be use with a for loop to find all the property in an object.
  • Properties can be added to an object after the object is declared. The object's property can be changed in the same way.
  • Properties can be nested inside of other properties.
  • Objects can be passed in a function's parameter.
  • The "this" keyword is used to refer to the object itself and it lets the functions gain access the object's properties.
  • Object's create their own namespace and are useful in preventing bugs.
  • JSON is a popular light-weight means of storing data and is very useful in JavaScript.
  • Use parse() to convert a JSON string to a JavaScript object and use stringify() to convert a JavaScript object to JSON.
  • The math objects are use to represent serveral mathimatical constants and they immutable and unable to be changed.
  • There are serveral math methods that provide useful mathimatical calculations.
  • All angle measurements should be in radians and not degrees.
  • Watch out for issue with floating point calculations.
  • Use the date object for handling date and time.
  • A getter method is used to get a property that is private.
  • A setter method is used to change a property that is private.
  • A regular expression is a pattern that can be used to search strings for matches to the pattern.
  • A regular expression can be created with "/" or with the constructor.
  • You can use the test() function to see if a string passes.

Chapter 6 - JavaScript: Novice to Ninja

  • The Document Object Model is the way of representing objects on an HTML page we can use it to change the apperance of a page and make it a dynamic web page.
  • The DOM provides several methods that allow us to access any element on a page.
  • The document keyword is used to access the web page.
  • The getElementById() function is used to referrence an element with that id.
  • Every id on the web page should be unique.
  • The getElementsByTagName() function returns a list of elements that have that tag.
  • The getElementsByClassName() function returns a list of elements that have that class.
  • The querySelector() function returns a list of elements that have the CSS selector.
  • The childNodes property returns all the nodes that are children of an element.
  • The parentNode property returns the parent node of an element.
  • The textContent property returns the text in an element as a string.
  • The getAttribute() function returns the value of the attribute.
  • The setAttribute() function can change the value of the attribute.
  • The className and classList properties get all the classes assigned to an element.
  • The createElement() function is used to create an DOM object.
  • The appendChild() function is used to append a node as a child node to another node.
  • The insertBefore() function will place a new element before another element.
  • The removeChild() function removes an element from the page.
  • The innerHTML() function returns all the child elements of an element as a string of HTML.
  • Every element node has a style property.
  • The getComputedStyle() function will retrieve all the style information of an element.
  • It is much better practice to dynamically change the class of an element and keep the relevant styles for each class in a separate stylesheet.

Chapter 7 - JavaScript: Novice to Ninja

  • An event listener triggers when an event happens and can be use to start a program.
  • Event listener is a non-block approach to programming.
  • Inline event handlers are triggers that are imbedded in the code but they are not recommended for use.
  • Using an event listener is the best approach to event handling.
  • Whenever an event handler is triggered by an event, the callback function is called.
  • The target property returns a reference to the node that fired the event.
  • The mouseover event occurs when the mouse pointer is placed over the element to which the event listener is attached.
  • The mousemove event occurs whenever the mouse moves.
  • There are three keyboard events, keydown, keypress, and keyup.
  • The key property returns the key that was pressed.
  • Pressing the modifier keys such as Shift, Ctrl, and Alt will fire the keydown and keyup events, but not the keypress event as they don’t produce any characters on the screen.
  • The touchstart event occurs when a user initially touches the surface and the touchend event occurs when a user stops touching the surface.
  • The removeEventListener() function removes an event listener.
  • preventDefault() is a function of the event object that can be used inside the callback function to stop the default behavior happening.
  • When you click on an element, you actually click on all the elements it’s nested inside of.
